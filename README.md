# get_next_line

Файлы с суффиксом bonus можно не смотреть, они идентичны. \
Дескриптор каждого нового файла добавляется в односвязный список, вход в который хранится в статической переменной \
Это позволяет читать вразнобой сразу несколько файлов.

На вход необходимо подать дескриптор файла и указатель на строчку, в которую считается строка из файла. \
При компиляции с флагом -d необходимо указать BUFFER_SIZE - кол-во считываемых байт за раз. \
Функция вернет 1, если прочитала строчку \
-1 если случилась ошибка
0 если файл прочитан
